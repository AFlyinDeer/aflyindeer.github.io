Minecraft Mods

This page acts as a catch all for the mods I use in Minecraft. The mods listed are Fabric mods and will require the user to install Fabric into their game. This can be achieved by download the Fabric Loader or by using MultiMC. Fabric API is very important when working with Fabric mods. Do keep in mind that some of these mods may conflict. I've had issues when carpet mod is installed on the client in the past.

Another great resource is Vanilla Teaks. Here you can create and add custom Resource Packs, Data Packs, and Crafting Tweaks to your game.
